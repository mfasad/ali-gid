Как узнать ID заказа на AliExpress: где искать и как использовать

Зачем нужен ID заказа и где его применяют

Каждый заказ на AliExpress получает уникальный идентификатор — ID заказа (Order ID). Этот код из 16–20 цифр нужен не только для отслеживания посылки, но и для решения спорных ситуаций. Если продавец не выходит на связь, товар потерялся или пришёл не тот — без номера заказа подтвердить покупку и инициировать возврат будет невозможно.

ID требуется в трёх ключевых случаях. Во-первых, для трекинга посылки на сайтах логистических компаний (например, Cainiao или Yanwen). Во-вторых, при общении с поддержкой AliExpress — без кода операторы не смогут найти ваш заказ в системе. В-третьих, ID нужен для оформления возврата или спора, если товар не соответствует описанию или не дошёл до адресата.

На практике пользователи часто путают ID заказа с трек-номером (номером для отслеживания). Разница в том, что трек-номер выдаётся логистической компанией и может меняться при передаче посылки между перевозчиками, а ID заказа остаётся неизменным от момента оплаты до закрытия сделки.

Где искать ID заказа на AliExpress

Номер заказа отображается в нескольких разделах платформы — как на сайте, так и в мобильном приложении. Вот что нужно сделать, чтобы его найти:

На сайте AliExpress

  1. Авторизуйтесь в своём аккаунте на aliexpress.com.

  2. Перейдите в раздел Мои заказы (иконка человека в правом верхнем углу → Заказы).

  3. Выберите нужный заказ из списка. ID будет указан под названием товара в формате «Номер заказа: 1234567890123456».

📋 Поиск ID на сайте

  1. Шаг 1. Откройте «Мои заказы»
  2. Шаг 2. Выберите нужный заказ
  3. Шаг 3. Найдите строку «Номер заказа» под названием товара
  4. Шаг 4. Скопируйте 16–20 цифр

В мобильном приложении

  1. Запустите приложение AliExpress и нажмите на иконку профиля внизу экрана.

  2. Перейдите в Мои заказы → выберите заказ, для которого нужен ID.

  3. Прокрутите страницу заказа вниз до блока Детали заказа. Номер будет указан рядом с надписью Order ID.

📝

В приложении ID заказа можно скопировать одним нажатием — нажмите на номер и выберите «Копировать» в появившемся меню.

В письме от AliExpress

После оформления заказа на вашу электронную почту (ту, что привязана к аккаунту) приходит письмо с подтверждением. В нём ID указан в самом верху, рядом с датой покупки. Если письмо потерялось, проверьте папку Спам или воспользуйтесь поиском по запросу AliExpress Order Confirmation.

Пошаговая инструкция: как проверить ID заказа

Если вы никогда раньше не искали номер заказа, следуйте этой инструкции. Она подходит для всех устройств — ПК, смартфонов и планшетов.

Шаг 1: Авторизация в аккаунте

Убедитесь, что вы вошли в тот аккаунт, с которого был сделан заказ. Если используете несколько почт или телефонов для регистрации на AliExpress, проверьте все профили. Важный момент: ID заказа привязан к аккаунту покупателя, а не к устройству.

Шаг 2: Поиск заказа в истории

В разделе Мои заказы заказы сортируются по статусам: Ожидает отправки, В пути, Доставлено. Если заказ старый (более 6 месяцев), воспользуйтесь фильтром по дате или поиском по названию товара.

⚠️

Заказы старше 18 месяцев автоматически архивируются и не отображаются в основном списке. Чтобы их найти, нажмите Архив заказов внизу страницы.

Шаг 3: Копирование или запись ID

Когда вы нашли нужный заказ, скопируйте ID одним из способов:

  • На сайте: выделите номер мышью и нажмите Ctrl+C (или правой кнопкой → Копировать).
  • В приложении: нажмите на номер заказа → в меню выберите Копировать.
  • На телефоне: сделайте скриншот экрана с ID (если копирование недоступно).

Шаг 4: Проверка корректности номера

Убедитесь, что скопированный ID соответствует формату:

  • Содержит только цифры (без букв, пробелов или символов).
  • Длина — от 16 до 20 знаков.
  • Начинается с цифр 1–9 (не с нуля).
💡

Если номер короче 16 символов, возможно, вы скопировали не ID заказа, а артикул товара (SKU) или трек-номер. Вернитесь в раздел Детали заказа и проверьте ещё раз.

Нюансы и подводные камни

Даже если вы нашли ID заказа, могут возникнуть сложности. Вот на что обратить внимание:

Заказ сделан через «Гостевую корзину»

Если покупка была оформлена без регистрации (через функцию Гостевая корзина), ID заказа привязан к email, указанному при оформлении. В этом случае:

  1. Проверьте почту, на которую приходило письмо с подтверждением.
  2. Если письма нет, попробуйте восстановить доступ через aliexpress.com/forgot_password, указав этот email.

ID не отображается в приложении

Иногда в мобильной версии AliExpress номер заказа скрыт из-за ошибки отображения. Решения:

  • Обновите приложение до последней версии.
  • Очистите кэш приложения в настройках телефона.
  • Попробуйте открыть заказ через браузер (в мобильной или десктопной версии сайта).

Заказ объединён с другими покупками

Если вы оформили несколько товаров в одном заказе (например, у одного продавца), у них будет общий ID. Чтобы найти номер конкретного товара:

  1. Откройте объединённый заказ.
  2. Нажмите на стрелочку рядом с нужным товаром.
  3. В открывшемся окне будет указан Order Item ID (уникальный номер позиции в заказе).
Что такое Order Item ID?

Это дополнительный идентификатор, который присваивается каждому товару в объединённом заказе. Он нужен, если вы хотите вернуть или заменить только один из нескольких товаров.

Сроки хранения информации о заказах

AliExpress хранит данные о заказах в личном кабинете ограниченное время. Вот актуальные лимиты:

Статус заказа Срок хранения в «Моих заказах» Архивируется?
Активный (в обработке, в пути) До завершения сделки Нет
Завершённый (доставлен, оценён) 18 месяцев Да
Отменённый или возвращённый 6 месяцев Да
⚠️

Правила хранения заказов могут изменяться. Актуальную информацию уточняйте в Пользовательском соглашении AliExpress.

Типичные ошибки при поиске ID заказа

Ошибки при работе с ID заказа ведут к задержкам в решении споров или невозможности отследить посылку. Вот самые распространённые:

  • Путают ID заказа с трек-номером. Трек-номер начинается с букв (например, LP123456789CN), а ID — только цифры. Если вы введёте трек-номер вместо ID при обращении в поддержку, вам откажут в помощи.
  • Копируют неполный номер. Иногда пользователи захватывают только часть ID (например, последние 8 цифр). Полный номер должен содержать все символы без обрезки.
  • Используют ID из другого заказа. При объединённых покупках легко перепутать номера. Всегда проверяйте, что ID соответствует нужному товару.
  • Не сохраняют ID заранее. Если заказ архивируется или удаляется, восстановить его номер будет сложно. Рекомендуется фиксировать ID сразу после покупки (например, в заметках телефона).

Когда ID заказа не помогает: альтернативные решения

Бывают ситуации, когда даже при наличии ID заказа решить проблему не удаётся. Например, если продавец не реагирует на сообщения, а поддержка AliExpress просит дополнительные доказательства. В таких случаях:

Если потерялся трек-номер: обратитесь к продавцу через чат на странице товара. Напишите: «Hello! I lost the tracking number for order ID [ваш номер]. Could you resend it?». Большинство продавцов присылают трек в течение 24 часов.

Если заказ не отслеживается: проверьте статус на альтернативных трекерах, например, 17track.net или parcelsapp.com. Иногда логистические компании обновляют данные с задержкой.

Если нужна помощь поддержки: прикрепите к обращению не только ID заказа, но и:

  • скриншот оплаты (из истории банковских операций);
  • фото товара (если речь идёт о браке или несоответствии);
  • скриншот переписки с продавцом.
